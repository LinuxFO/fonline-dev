# AGENTS.md FOnline Dev

This repository contains FOClassic, PReloaded and FOnline 2258 source codes.

## Tips

- All documentation files, reports, notes are Markdown files.
- `fonline-dev/ai/docs` directory contains docs about FOnline development.
- `fonline-dev/ai/migration` directory contains migration files and FOnline 2258 source code.
- `fonline-dev/ai/migration/fonline-2258` directory contains FOnline 2258 source code (config, defines, maps, scripts etc.).
- `fonline-dev/foclassic` directory contains FOClassic (FOnline Engine) source code.
- `fonline-dev/PReloaded` directory contains PReloaded source code and migrated FOnline 2258 files (config, defines, maps, scripts etc.).
- Do not modify `fonline-dev/foclassic` source code.
- Create general helper scripts in `fonline-dev/ai/scripts` directory.
- Create migration helper scripts in `fonline-dev/ai/migration/scripts` directory.
- FOnline script is written in AngelScript, file extension is `.fos`
- For each migration iteration create plan (tasks), and implementation log report (what was done, progress status).
- Before each migration iteration review changes.
